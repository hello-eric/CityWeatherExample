这是一个简单天气应用,但是包含了一个完整的APP框架,需要的朋友可以拿去借鉴一下,不足的地方请不吝指教。
	
	1.项目基本为MVP架构
	2.其中包含单例、工厂、多态等设计模式应用实战
	
使用到第三方库
	
	1. JSON解析库 gson-1.7.1.jar 
	2. 趋势图库 mpandroidchartlibrary-2-1-3.jar
	3. 图片下载库 universal-image-loader-1.9.2.jar
	4. 网络请求库 Volly
